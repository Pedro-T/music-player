# Simple Music Player


## Purpose
I want a simple music player that integrates youtube-dlp for adding to the library, primarily to play various long chill work/productivity tracks without having to sit on Youtube and stream the pointless video component.

## Tech
- Godot for the UI
- C#
- Youtube-DLP integration for downloads

## MVP1 Plan
- Able to play tracks
- Volume control
- Pause / Play / Next / Previous controls
- auto move to next track (shuffle only) based on a target music directory

## MVP2 Plan
- playlist editing
- playlist shuffle / linear
- nicer UI
